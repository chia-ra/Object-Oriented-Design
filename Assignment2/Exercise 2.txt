The main differences between int and Integer:
- int stores integer values into memory, Integer converts an int to object (and vice versa)
- int is less flexible than Integer:
	Integer provides functionalities which can be used on an int value, such as:
	-toString() and variants of, valueOf,  rotateLeft() or rotateRight(),  etc.
	-additionally, Integer allows to convert to other value types (ex. byte, short, int)
	-Integer also allows strings to be assigned or converted to int values using parsing 
	 or constructors, whereas int primitive type does not, even with casting.
	-Wih Integer class, you can use it in collections with Object reference or Generics.
- however, int data type takes 4 times fewer bits than Integer class to store an int. (32 vs. 128 bits)
- specific functionalities you can use with int only are operators like +, -, /, *, ++, --, ==. !=, etc.
  this is because Integer objects made with the Integer class are immutable and cannot be changed once created.
